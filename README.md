# Devesh Kumar Mandal — Portfolio

3D Data Scientist Portfolio built with React + Vite + Three.js + Framer Motion.

## Tech Stack
- React 19 + Vite
- Three.js — animated neural network particle canvas
- Framer Motion — smooth entrance animations
- Space Mono + Syne fonts
- Deep space / cyan neon theme

## Local Development
```
npm install
npm run dev
```

## Build for Production
```
npm run build
```

## Deploy on Vercel (Recommended — Free)
1. Push this folder to GitHub
2. Go to vercel.com → New Project → Import your repo
3. Framework: Vite (auto-detected)
4. Click Deploy — live in ~60 seconds!

## Deploy on Netlify (Free)
1. Push to GitHub
2. netlify.com → Add New Site → Import from Git
3. Build command: npm run build
4. Publish directory: dist
5. Click Deploy

## Customise
Edit src/data/portfolio.js to update all your info — everything flows from that one file.
